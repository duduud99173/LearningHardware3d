这是一个基于 Direct3D11 和 Win32 API 构建的轻量级渲染引擎学习项目。它主要作为我在学习图形编程过程中的实践平台，旨在深入理解 D3D11 渲染管线的工作方式，并在此基础架构上进行其他图形算法的探索。
本项目在很大程度上是基于 **planetchili** 在 GitHub 上的 **[Hardware 3D (hw3d)](https://github.com/planetchili/hw3d)** 系列教程和提供的引擎架构进行学习和构建的。
* **致谢 planetchili：** 衷心感谢 planetchili 提供了一个清晰、优秀的 D3D11 学习框架和基础引擎代码。项目的初始化、核心的渲染器架构、Win32 封装以及纹理贴图和 Blinn-Phong 光照的基础实现均是在学习其教程的过程中完成。

本项目在继承了基础渲染架构的同时，探索并实现了以下图形渲染功能：
* **法线贴图**
* **体积云**
